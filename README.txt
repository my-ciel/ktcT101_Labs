kt cloud Essential Course Lab
-------------------------------------------

1. OVERVIEW
This project is a customized web template designed for the "kt cloud Essential Course." 
The primary goal is to provide a hands-on lab environment where students can practice 
deploying a web server on kt cloud infrastructure.

2. DIRECTORY STRUCTURE
- index.html: The main landing page for the lab.
- assets/: Contains CSS and JS files for styling and functionality.
- images/: Stores images and icons used throughout the site.

3. HOW TO DEPLOY
- Step 1: Create a Virtual Machine (VM) on the kt cloud console.
- Step 2: Install a web server (Nginx or Apache).
- Step 3: Upload all files in this directory to the web server's root directory (e.g., /var/www/html).
- Step 4: Access the web service via the VM's Public IP.
