kt cloud 기반 가상 서버 및 웹 서비스 배포 실습

본 실습은 kt cloud 인프라를 활용하여 가상 서버를 생성하고,
웹 서버 소프트웨어를 설치하여 웹 서비스를 공개하는 과정을 학습합니다.

Compute

높은 신뢰도와 안정성을 가진
최고급 클라우드 인프라 서비스를 제공합니다.

Networking

빠르고 안정적인 고품질의 네트워크 서비스를 제공하여
웹 서비스의 접근 제어와 보안을 설정합니다.

Container

kt cloud 환경에서 다양한 Container 서비스를
빠르고 간편하게 제공합니다.

Security

데이터와 애플리케이션을 효과적으로 보호하는
보안/관리 서비스를 제공합니다.
